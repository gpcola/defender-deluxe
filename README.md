# Defender Deluxe

A modern browser remake inspired by the 1981 arcade classic **Defender**.

- Pure HTML5 Canvas (no assets, no libraries)
- Procedural WebAudio SFX
- CRT scanlines, glow, particles, screen shake
- **Daily Challenge seed**
- **Leaderboard via GitHub Issues (no backend)**

## Play
https://gpcola.github.io/defender-deluxe/

## Controls
- **← / →** rotate
- **↑** thrust
- **Space** fire
- **Shift** hyperspace
- **B** smart bomb
- **P** pause
- **S** submit score (on Game Over)

## Leaderboard
Scores are submitted as GitHub Issues labeled `score`.
